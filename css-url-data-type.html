সিএসএস URL

সিএসএস এ <URL> বলতে বোঝায় কোনো কিছুর রিসোর্স। যেমন কোনো ইমেজ, ফন্ট এদের রিসোর্স ফাইল বোঝাতে ইউআরএল ব্যবহার হয়। ইউআরএল প্রচুর সিএসএস প্রোপার্টিতে ব্যবহার করা হয়। যেমন background-image, cusor, list-style.

<url> ডাটা টাইপ ব্যবহার করা হয় url() ফাংশন দিয়ে। প্রথম বন্ধনীর ভিতরে কোটেশন ছাড়া, অথবা সিঙ্গেল/ডাবল কোটেশন দিয়ে মার্ক করা থাকে।

ইউআরএল সম্পর্কে এইচটিএমএল ওয়েব অ্যাপে বিস্তারিত জানতে পারবে।

নিচে কিছু উদাহরণ দেয়া হলো:

.header {
    background: url('backimage.jpg');
}
//এখানে সিঙ্গেল কোটেশন ব্যবহার করা হয়েছে।

ul {
  list-style: square url("bullet.png");
}
//এখানে ডাবল কোটেশন ব্যবহার করা হয়েছে।

ul {
  list-style: square url(bullet.png);
}
// এখানে কোনো কোটেশন মার্ক ব্যবহার হয় নাই।
