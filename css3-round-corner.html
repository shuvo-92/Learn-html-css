Css3 Rounded corner

CSS 3 তে আমরা এলিমেন্ট কর্ণারকে Rounded করতে পারি।

border-radius সিএসএস প্রোপার্টি দিয়ে যেকোনো এলিমেন্টের বর্ডারের কোণা রাউন্ড শেপ করা যায়।

    যে সকল ভ্যালু ব্যবহার করা যাবে:
  • <length>
  • <percentage>

উদাহরণ:

				
div {
border-radius: 10px;
width: 200px;
height: 200px;

	}
