ফন্ট ফেইস(@font-face) সিএসএস নিয়ম

আপনি আপনার যেকোনো ফন্ট ব্যবহার করতে পারেন @font-face ব্যবহার করে।

    এর জন্য নিচের নিয়ম অনুসরণ করুন:
  • font-family: আপনি ফন্টকে যে নামে ব্যবহার করতে চান তা দিন।
  • src:ফন্ট যেখানে আছে তা url ফাংশন দিয়ে আনুন। যেকোনো ফোল্ডার বা লোকেশন থেকে আনতে ব্যবহার করুন।।

@font-face{ font-family:myFont; src:url('folder1/myFont.ttf'); }
