সিএসএস স্যুডো এলিমেন্টস | Css pseudo elements

সিএসএস স্যুডো এলিমেন্ট হলো একটি কী-ওয়ার্ড। যার মাধ্যমে একটি সিলেক্টর ব্যবহার করে কোনো এলিমেন্টের নির্দিষ্ট অংশ স্টাইল করা যায়।


Syntax কিভাবে ব্যবহার হয়:

selector::pseudo-element {
    property:value;
}


সিএসএস-এ কি কি ধরনের স্যুডো এলিমেন্ট ব্যবহার নিচে তা জানবো:
  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection

    • ::after স্যুডো এলিমেন্ট

    ::after স্যুডো এলিমেন্ট এমন একটি স্যুডো এলিমেন্ট তৈরি করে যেটি কোনো এলিমেন্টের last child। মানে কোনো এলিমেন্টের একদম শেষের দিকে ডিজাইন করতে বা এর জন্য ব্যবহার হয়।

    সাধারণত নির্দিষ্ট কোনো এলিমেন্টের সাজসজ্জা দিতে content প্রোপার্টি দ্বারা এটি ব্যবহার করা হয়।
    যেগুলো ডিফল্টভাবে একটি ইনলাইন এলিমেন্ট তৈরি করে।

    .true::after {
    content: "-> True";
    color: green;
    }

    .false::after {
    content: "-> False";
    color: red;
    }

    ::before স্যুডো এলিমেন্ট

    ::before স্যুডো এলিমেন্ট এমন একটি স্যুডো এলিমেন্ট তৈরি করে যেটি কোনো এলিমেন্টের first child। মানে কোনো এলিমেন্টের একদম প্রথম দিকে ডিজাইন করতে বা এর জন্য ব্যবহার হয়।

    সাধারণত নির্দিষ্ট কোনো এলিমেন্টের সাজসজ্জা দিতে content প্রোপার্টি দ্বারা এটি ব্যবহার করা হয়।
    যেগুলো ডিফল্টভাবে একটি ইনলাইন এলিমেন্ট তৈরি করে।

    .true::before {
    content: "-> True";
    color: green;
    }

    .false::before {
    content: "-> False";
    color: red;
    }

    ::first-letter স্যুডো এলিমেন্ট

    ::first-letter স্যুডো এলিমেন্ট টি নির্দিষ্ট ব্লক লেভেল এলিমেন্টের প্রথম লাইনের প্রথম অক্ষরের স্টাইল দিতে ব্যবহার করা হয়।

    যেসকল সিএসএস প্রোপার্টি তে ব্যবহার করা যায়।

  • সকল font প্রোপার্টি।
  • সকল background প্রোপার্টি।
  • সকল margin প্রোপার্টি।
  • সকল padding প্রোপার্টি।
  • সকল border প্রোপার্টি।
  • সকল color প্রোপার্টি।
  • word-spacing letter-spacing text-transform line-height text-shadow text-decoration text-decoration-color text-decoration-line text-decoration-style vertical-align প্রোপার্টি।।

    • p::first-letter {
    text-decoration: overline;
    font-size: 250%;
    color: red;
    }

    ::first-line স্যুডো এলিমেন্ট

    ::first-line স্যুডো এলিমেন্ট টি নির্দিষ্ট ব্লক লেভেল এলিমেন্টের প্রথম লাইন কে স্টাইল দিতে ব্যবহার করা হয়।

    যেসকল সিএসএস প্রোপার্টি তে ব্যবহার করা যায়।

  • সকল font প্রোপার্টি।
  • সকল background প্রোপার্টি।
  • color প্রোপার্টি।
  • word-spacing letter-spacing text-transform line-height text-shadow text-decoration text-decoration-color text-decoration-line text-decoration-style vertical-align প্রোপার্টি।।

    • p::first-letter {
    color: #f8f9f9;
    background-color: #173459;
    }

    ::selection স্যুডো এলিমেন্ট

    ওয়েবাসাইট, লেখায়, ডকুমেন্টে ব্যবহাকারী যখন কোনো কিছু সিলেক্ট করে তখন সিলেক্ট করা অংশ আমরা যেভাবে দেখাতে চাই বা আমরা যেরকম স্টাইল দিতে চাই তা এই ::selection স্যুডো এলিমেন্টের মাধ্যমে করা হয়।


    যেসকল সিএসএস প্রোপার্টি তে ব্যবহার করা যায়:

  • color
  • caret-color
  • background-color
  • cursor
  • outline প্রোপার্টি গুলো।
  • text-decoration এবং এর অধিভুক্ত সব প্রোপার্টি।
  • text-shadow
  • text-emphasis-color

    • p::selection {
    color: red;
    background-color: silver; //লেখাটি সিলেক্ট করে দেখো কেমন হয়।
    }

    Note: এই ডেমো IOS এ কাজ করবে না।
