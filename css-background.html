সিএসএস ব্যাকগ্রাউন্ড

এলিমেন্ট গুলোর ব্যাকগ্রাউন্ড কালার বা ইমেজ দিয়ে ফিল দিয়ে থাকে, ক্লিপড বা রিসাইজ করা যায় এমন থাকে আরো অন্যান্য কাজ করা যায় ব্যাকগ্রাউন্ড নিয়ে।


সিএসএস ব্যাকগ্রাউন্ড প্রোপার্টি:

  • backgroun-color
  • background-image
  • backgroun-repeat
  • backgroun-attacent
  • backgroun-position

    • ব্যাকগ্রাউন্ড কালার

    backgroun-color: প্রোপার্টি দিয়ে আমরা যেকোনো এলিমেন্টের ব্যাকগ্রাউন্ড কালার পরিবর্তন করতে পারি।
    এর ভ্যালু হচ্ছে সিএসএস কালার বা সিএসএস৩ কালার এর ভ্যালিড যেকোনো কালার।
    পুরো পেজের ব্যাকগ্রাউন্ড পরিবর্তন করতে:

    body {
      background-color: yellow;
    }

    যেকোনো এলিমেন্টের ব্যাকগ্রাউন্ড পরিবর্তন:

    h1 {
    background-color: yellow;
    }
    p { background-color: pink;
    }
    div {
    background-color: gray;
    width: 200px;
    height: 200px;
    }

    ব্যাকগ্রাউন্ড ইমেজ

    backgroun-image প্রোপার্টি যেকোনো এলিমেন্টের ব্যাকগ্রাউন্ডে ছবি দিতে ব্যবহার করা হয়।
    এর সঠিক ভ্যালু হচ্ছে url("image.jpg")। সিঙ্গেল কোটেশন বা কোটেশন না দিলেও কাজ করবে। যেমন url('image.jpg') এবং url(image.jpg)।
    কোটেশন এর ভিতরে যা থাকে তা হচ্ছে ফাইল প্যাথ। অর্থাৎ ছবিটি কোথায় আছে তা বলে দেয়া।

    body {
    background-image: url("images/clouds-bg.jpg");
    }

    নোট: এমন কোনো ব্যাকগ্রাউন্ড ছবি বা কালার ব্যবহার করা যাবে যা পেজের টেক্সট পড়তে, দেখতে সমস্যা করে। সবসময় সুন্দর এবং ভালো কালার ব্যবহার করতে হবে।

    ব্যাকগ্রাউন্ড ইমেজ রিপিট

    ব্যাকগ্রাউন্ড ইমেজ অটোমেটিক ভাবে রিপিট হতে থাকবে সমান্তরাল/হরাইজন্টাল/(x-অক্ষ) এবং উলম্ব/ভার্টিকাল/(y-অক্ষ) এর দিকে।
    নিচের উদাহরণ টি দেখো:

    body {
    background-image: url("images/small-sunflower.png");
    }

    শুধু একদিকে সমান্তরাল/হরাইজন্টাল/(x-অক্ষ) বা উলম্ব/ভার্টিকাল/(y-অক্ষ) দিকে ব্যাকগ্রাউন্ড ইমেজ কে রিপিট করতে background-repeat সিএসএস প্রোপার্টি ব্যবহার করা হয়।
    সমান্তরাল/হরাইজন্টাল/(x-অক্ষ) এর দিকে ইমেজ রিপিট:

    body {
    background-image: url("images/small-sunflower.png");
    background-repeat: repeat-x;

    উলম্ব/ভার্টিকাল/(y-অক্ষ) এর দিকে ইমেজ রিপিট:

    body {
    background-image: url("images/small-sunflower.png");
    background-repeat: repeat-y;
